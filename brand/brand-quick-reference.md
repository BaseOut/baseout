# Baseout Brand Quick Reference

## Logo Usage
- **Primary**: Horizontal logo on light backgrounds
- **Minimum size**: 120px wide (digital), 1.5" (print)
- **Clear space**: Equal to height of "baseout" text
- **File formats**: SVG (web), EPS (print), PNG (fallback)

## Color Palette
```
Primary: #2D5A5A (Baseout Teal)
Secondary: #4A7C7C (Teal Light), #1A3434 (Teal Dark)
Accent: #D17A3A (Peak Orange)
Text: #2C3E50 (Charcoal), #5D6D7E (Slate Gray)
Background: #F4F6F8 (Light Gray), #FFFFFF (White)
```

## Typography
- **Primary**: Inter (400, 500, 600, 700)
- **Display**: Outfit (600, 700, 800)
- **Code**: Source Code Pro (400, 500, 600)
- **Minimum body text**: 16px

## Spacing System
```
Base unit: 8px
Micro: 4px | XS: 8px | S: 16px | M: 24px | L: 32px | XL: 48px | XXL: 64px
```

## Grid System
- **Columns**: 12-column grid
- **Gutters**: 24px
- **Max width**: 1200px
- **Breakpoints**: 480px, 768px, 1024px, 1440px

## Voice & Tone
- **Personality**: Professional yet approachable, trusted expert
- **Tone**: Clear, confident, innovative but grounded
- **Avoid**: Jargon without explanation, overpromising

## Icon Style
- **Grid**: 24px base
- **Stroke**: 2px weight
- **Style**: Outline primary, rounded corners
- **Colors**: Teal primary, Slate Gray secondary

## Messaging
**Primary**: "Build data infrastructure that scales"
**Value Prop**: "Transform complex data challenges into competitive advantages"
