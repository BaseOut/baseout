## Color Usage Guidelines

### Accessibility Standards
- Maintain WCAG 2.1 AA compliance (4.5:1 contrast ratio)
- Test color combinations for colorblind accessibility
- Use sufficient color contrast for text readability
- Don't rely solely on color to convey information

### Brand Hierarchy
1. **Primary**: Baseout Teal (70% of usage)
2. **Secondary**: Supporting colors (20% of usage)
3. **Accent**: Peak Orange for highlights (10% of usage)

### Digital Applications
- Use hex values for web development
- Ensure colors render consistently across devices
- Consider dark mode alternatives
- Test on various screen types and lighting conditions

### Print Applications
- Use CMYK values for professional printing
- Account for color variation in different paper types
- Test colors with chosen printing methods
- Consider spot color printing for brand consistency

## Color Combinations

### Approved Combinations
- Baseout Teal + Pure White
- Charcoal + Background Gray
- Peak Orange + Pure White
- Data Blue + Light Gray
- Mountain Green + Background Gray

### Avoid These Combinations
- Baseout Teal + Mountain Green (too similar)
- Peak Orange + Error Red (competing warm colors)
- Slate Gray + Light Gray (insufficient contrast)
- Any combination below 3:1 contrast ratio
