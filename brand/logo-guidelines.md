# Logo Guidelines

## The Baseout Logo

The Baseout logo is the cornerstone of our brand identity. It combines a distinctive geometric symbol with clean, modern typography to create a memorable and professional mark.

## Logo Components

### Symbol
- **Layered mountain icon**: Represents data layers, scalability, and reaching peak performance
- **Geometric container**: Suggests structured, systematic approach
- **Rounded corners**: Adds approachability while maintaining professionalism

### Logotype
- **"baseout"**: Clean, sans-serif typeface
- **Lowercase styling**: Approachable and modern
- **Consistent spacing**: Maintains readability and balance

## Primary Logo Versions

### 1. Horizontal Logo (Primary)
- Default version for most applications
- Symbol to the left, logotype to the right
- Optimal for headers, business cards, and wide layouts

### 2. Stacked Logo
- Symbol above logotype
- Use when horizontal space is limited
- Ideal for social media profiles, app icons

### 3. Symbol Only
- Use when brand is well-established
- Minimum size: 24px digital, 0.5" print
- App icons, favicons, watermarks

### 4. Logotype Only
- Text-only version
- Use in copy-heavy contexts
- When symbol reproduction is challenging

## Logo Variations

### Color Treatments
- **Primary**: Baseout Teal on white/light backgrounds
- **Reversed**: White on dark backgrounds
- **Monochrome**: Single color applications
- **Grayscale**: When color is not available

### Size Specifications

#### Minimum Sizes
- **Horizontal Logo**: 120px wide (digital), 1.5" wide (print)
- **Stacked Logo**: 80px wide (digital), 1" wide (print)
- **Symbol Only**: 24px (digital), 0.5" (print)

#### Optimal Sizes
- **Web headers**: 160px wide
- **Business cards**: 2" wide
- **Presentations**: 200px wide
- **Large format**: Scale proportionally

## Clear Space Requirements

Maintain clear space around the logo equal to the height of one "baseout" letter.

## Usage Do's and Don'ts

### ✅ DO
- Use approved logo files
- Maintain aspect ratio
- Ensure sufficient contrast
- Use on approved backgrounds
- Keep clear space requirements
- Use high-resolution files for print

### ❌ DON'T
- Stretch or distort the logo
- Change colors outside approved palette
- Add effects (drop shadow, glow, etc.)
- Place on busy backgrounds
- Use low-resolution files
- Separate symbol from logotype arbitrarily
- Change font or modify lettering
- Place too close to other elements

## File Formats

### Digital Applications
- **SVG**: Scalable vector (preferred for web)
- **PNG**: Transparent background
- **JPG**: Solid background only

### Print Applications
- **EPS**: Vector format for professional printing
- **PDF**: High-resolution for various print needs
- **PNG**: High-resolution (300 DPI minimum)

## Background Applications

### Light Backgrounds
- Use primary Baseout Teal logo
- Ensure minimum 70% contrast ratio
- White, light gray, or subtle brand colors

### Dark Backgrounds
- Use white or light logo versions
- Ensure logo stands out clearly
- Dark navy, black, or dark brand colors

### Photography
- Use white logo with subtle drop shadow
- Ensure background area is not busy
- Consider logo placement carefully

## Special Considerations

### Digital Applications
- Use SVG format when possible for crisp rendering
- Ensure logo scales properly on different screen sizes
- Test visibility on various devices and browsers

### Print Applications
- Use vector formats for scalability
- Ensure sufficient resolution (300 DPI minimum)
- Consider paper color and texture

### Merchandise
- Simplified versions may be needed for embroidery
- Consider single-color applications
- Test logo on different material colors

## Logo Approval Process

Before using the logo in new applications:
1. Review these guidelines
2. Select appropriate logo version
3. Ensure proper sizing and clear space
4. Verify color accuracy
5. Get approval for non-standard applications
